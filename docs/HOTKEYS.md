# Hotkeys

## General

- `Click`
  Select an object, or place the previewed asset in placement mode.
- `Esc`
  Leave placement mode and clear the current preview.
- `Delete`
  Delete the current selection.
- `Enter`
  Place the current previewed asset.
- `Ctrl + S`
  Save the scene to local storage.

## Undo / Redo

- `Ctrl + Z`
  Undo.
- `Ctrl + Shift + Z`
  Redo.
- `Ctrl + Y`
  Redo.

## Placement / Transform

- `R`
  Rotate the active target by the current rotation step on the selected axis.
- `Shift + A`
  Enter placement mode for the current active asset, if one is selected in the asset browser.
- `Shift + D`
  Duplicate the current selection.
- `Shift + F`
  Frame the current selection.

## Camera-Aligned Nudge

- `W`
  Move the current selection forward relative to the camera, constrained to one world axis.
- `S`
  Move the current selection backward relative to the camera, constrained to one world axis.
- `A`
  Move the current selection left relative to the camera, constrained to one world axis.
- `D`
  Move the current selection right relative to the camera, constrained to one world axis.

## Scene Tree Selection

- `Ctrl` or `Cmd` + click
  Add or remove one item from the current selection.
- `Shift` + click
  Select a visible range in the current Scene Tree order.
